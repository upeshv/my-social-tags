== Description ==
* Social Open Graph plugin allows you to add metadata to your page. Which turn your web pages into graph objects. This will place additional <meta> tags in the <head> of your web page.

* Whenever you will create a new post page it will automatically turn your web pages into graph objects with basis information. For example : title,url,image.

* You will have an options to add more metadata to your web page. 

* This enables any web page to become a rich object in a social graph. For instance, this is used on any social sites to allow any web page to have the same functionality as any other object on the sites.

* While many different technologies and schemas exist and could be combined together, there isn't a single technology which provides enough information to richly represent any web page within the social graph. This builds on these existing technologies and gives developers one thing to implement. 

== Installation ==

* Install this my-social-tags plugin and activate it, after activating. Your post page will automatically turn your web pages into graph objects with basis information. For example : title,url,image.

* On every post page you will find an option to add more Metadata to your web page.
